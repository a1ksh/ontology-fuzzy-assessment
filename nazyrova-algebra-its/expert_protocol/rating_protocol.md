# Expert rating protocol

## Raters
Two independent teachers (>= N years of algebra teaching). They did NOT see each
other's ratings. Relationship to ontology construction: [independent / same team]
(state clearly; if same, note as a limitation).

## Scale (Table 3)
Four-level ordinal scale per concept and per link:
- l0 - does not command
- l1 - partially commands (reproduces definition, cannot apply)
- l2 - reproductive (standard transformations by example)
- l3 - productive (transfers to new problem types)

## Procedure
1. For each learner, each rater assigns a level to every concept and every link
   (48 + 86 = 134 judgements/learner).
2. Inter-rater agreement = weighted Cohen's kappa (reported: 0.87, upper bound).
3. Reference profile = reconciliation by discussion; residual disagreements
   resolved by a third senior teacher.
